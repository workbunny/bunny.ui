API

JSON属性

属性 默认值 必须 描述 版本
code 0 是 提示类型:0-默认,1-绿色,2-黄色,3-红色,4-蓝色
msg '' 是 提示消息
time 3 否 提示显示时间,单位秒
anim 'scale' 否 提示动画:scale-缩放,left-左移,right-右移,up-上移,down-下移

Js用法

bny.alert(msg, code, anim, time)

属性 默认值 必须 描述 版本
msg '' 是 提示消息
code 0 是 提示类型:0-默认,1-绿色,2-黄色,3-红色,4-蓝色
anim 'scale' 否 提示动画:scale-缩放,left-左移,right-右移,up-上移,down-下移
time 3 否 提示显示时间,单位秒

公共类

公共类是 bunny.css 中并不以组件形式存在的公共 class 选择器，而又能用于任何地方。

普通

类名 描述
clearfix 清除浮动
fl 左浮动
fr 右浮动
tl 左对齐
tr 右对齐
tc 居中对齐
ellipse 省略号
inline 行内块

内外边距

类名 描述
p-12 12px 内边距
p-24 24px 内边距
p-36 36px 内边距
m-12 12px 外边距
m-24 24px 外边距
m-36 36px 外边距

字体大小

类名 描述
fs-16 16px 字体大小
fs-18 18px 字体大小
fs-20 20px 字体大小
fs-24 24px 字体大小
fs-32 32px 字体大小

动画

类名 描述
bny-anim-scale 放大
bny-anim-scaleOut 缩小
bny-anim-rotate 一直旋转
bny-anim-left 左滑入
bny-anim-leftOut 左滑出
bny-anim-right 右滑入
bny-anim-rightOut 右滑出
bny-anim-up 上滑入
bny-anim-upOut 上滑出
bny-anim-down 下滑入
bny-anim-downOut 下滑出
